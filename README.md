# Цены на предметы (со схронов,домов и т.д) на аукционе в игре Stay out ( Stalker Online )

**So-Price** — интерактивная база цен предметов вселенной S.O.
- Актуальные прайсы и ориентиры для торгов на аукционе/доске
- Поиск по названию, подсветка совпадений
- История и динамика (планируется)


## Стек
- Static (GitHub Pages), Vanilla JS + React UMD
- JSON-источник данных, lazy-загрузка, локальное кэширование корзины

## Развёртывание
- `main` → GitHub Pages
- Путь данных: `/data/data.json`
- Иконки: `/icons/*.png` (имена соответствуют `icon_local_path`)
